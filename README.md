# Web Development First Project
## Purpose
The purpose of this website is to give some information to the user about the Muppets. 
Users can explore information about the muppet movies and some of the main characters.
## Key Features
+ Navigation bar can be used to navigate from the home page to the character page and the movie page
+ "Start the show" button shows the user a picture of the Muppets cast from the Muppet Show intro song
+ Table of movie information gives movie title (with link to wikipedia), year, cover image, director, and a yes or no of whether Hannah and Jes have seen it
+ Table of Character information with an image and a famous quote from the character
